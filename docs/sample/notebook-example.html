.ipynb 파일 활용#

Jupyter Book에선 .ipynb파일 또한 지원합니다. 아래와 같이 코드를 입력하고, 그에 대응하는 출력물을 함께 웹페이지로 구성 가능합니다.

import matplotlib.pyplot as plt

plt.plot([3,1,2,1,3])

[<matplotlib.lines.Line2D at 0x21ff8e508e0>]

공식 홈페이지를 참고하여 interactive한 시각화도 가능합니다.
